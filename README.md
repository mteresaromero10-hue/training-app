# My Training Calendar

Personal training app — 5 session program with exercise tracker.

## Sessions
- Lower Body A (hip hinge + posterior chain)
- Lower Body B (quad dominant + glute med)
- Upper Body A (pull focus)
- Upper Body B (push + shoulders)
- Cardio + Core

## Features
- Tap a session to start it
- Check off exercises as you go
- Calendar logs automatically
- 4-week progressive overload plan
- Push-up and pull-up progressions
- Cycle-aware training guide
- Recovery day options
